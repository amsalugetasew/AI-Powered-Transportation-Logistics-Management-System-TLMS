## 📊 API Endpoints Available

### 1. **Chat Endpoint**
```
POST /api/ai/chat/
```
- Send natural language queries
- Returns AI analysis and formatted response
- Creates/updates conversation

### 2. **Conversation History**
```
GET /api/ai/history/
```
- Get all user conversations
- Returns last 20 conversations with messages

### 3. **Conversation Detail**
```
GET /api/ai/conversation/<id>/
```
- Get specific conversation with all messages

### 4. **File Download**
```
GET /api/ai/download/<message_id>/
```
- Download generated Excel/CSV/PDF files
- Returns file with appropriate content type

### 5. **Knowledge Base**
```
GET /api/ai/knowledge/
```
- Get knowledge base entries
- Filter by category (optional)

## 🔑 Next Steps - Getting Your Groq API Key

### Option 1: Use Groq (Recommended - FREE!)

1. Go to https://console.groq.com/
2. Sign up for a free account (no credit card required!)
3. Navigate to "API Keys" section
4. Click "Create API Key"
5. Copy the key (starts with `gsk_...`)
6. Open your `.env` file
7. Replace `your_groq_api_key_here` with your actual key:
   ```env
   GROQ_API_KEY=gsk_your_actual_key_here
   ```
8. Restart the backend server

### Why Groq?
- ✅ **FREE** tier with generous limits
- ✅ **Fast** inference (faster than OpenAI)
- ✅ **Powerful** models (Llama 3.1 8B & 70B)
- ✅ **No credit card** required for free tier
- ✅ **Easy** to use API

### Option 2: Use OpenAI (Alternative)
If you prefer OpenAI, you can modify `ai_assistant/analyzer.py` to use OpenAI's API instead.
